# NTU Swimming Pool Service

Real-time swimming pool status and social platform for NTU.

## Features
- **Weather Status**: Real-time lightning alerts via NEA API.
- **Community**: Verified NTU student/staff feed.
- **Mobile First**: Optimized for on-the-go usage.

## Setup
1. Create `.env` from `.env.example`.
2. Run `pip install -r requirements.txt`.
3. Run `python init_db.py` to create tables.
4. Run `flask run`.
